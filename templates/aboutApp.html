Working:



This is a Python software that encrypts and decrypts a file. When you want to ENCRYPT a file, just go to Encrypt Page and upload any file such as pdf or jpg or rar file. Now click on Upload button. Your file will be taken care of by using various algorithms and at last a KEY will be generated, you can DOWNLOAD the key using the download button on the SUCCESS page. You will need the Key to Decrypt the file whenever needed. Without the Key, the file cannot be retrieved.
Now, if you want to retrieve that file. Go to DECRYPT page, and upload the KEY that you downloaded. After successful operations using the Key, the file will be restored and you will be taken to the Success page where you can download the ORIGINAL FILE.
